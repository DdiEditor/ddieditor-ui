-------------------------------------------------------------------------------
What's New
-------------------------------------------------------------------------------
User interface
Item ID:    Description:
- Key bindings started. Open DDI3 and change between views, perspectives and 
editors. Key bindings:  
 * Next view: CTRL+1
 * Next perspective: CTRL+2 
 * Next editor: CTRL+TAB
 * Open DDI3: CTRL+O

- Changed behaviour of view create to if possible to expand only to the 2nd 
level.

- Changed behaviour of view refresh from expand all to keep state of 
expanded items.

- Added label on editor tabs if any labels asigned to the edit element.

- Variable question relation view with relation edit.

- Codes (value and category relation) added to Code Scheme Editor.

- Question Item editor added Name field edit. Name on question item now used as
label. 			

-------------------------------------------------------------------------------  
Application Infrastructure:

- Fixed Eclipse bug in treeviewer refresh.
