What's New
-------------------------------------------------------------------------------
User interface
Item ID:    Description:
DDIE-58 SPSS import: The user is now able to type the path to the sav and data file in the text input fields. The value is validated on CR.
DDIE-59 Question import: The user is now able to type the path to the wiki file in the text input field. The value is validated on CR.
- General for all import functions: If only one resource is defined (only one DDI Instance), this resource is preselected in the import pop-ups.
- Variable Question Relation view: View will now automatically propose a Variable/Question Item releation based on the variant number given in the wiki file.
- Wiki interface:
  a. Category Schemes are created.
  b. Categories are created.
  c. References from Question Item to Code Scheme - is created.
  d. Reference from Code (of Code Scheme) to Category - is created.
- SPSS import: Categories are no longer created.
