Uninstalling
-------------------------------------------------------------------------------
 - Remove installation directory e.g. C:\Programmer\ddieditor
 - Remove environment variable DDIEDITOR_HOME
 - Remove from environment added parameter from %PATH% variable: 
   %DDIEDITOR_HOME%\resources\native-libs
 - Uninstall DBXML
