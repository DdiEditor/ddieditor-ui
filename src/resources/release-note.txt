What's New
-------------------------------------------------------------------------------
User interface
Item ID:    Description:
 - Universes referenced by a Study Unit are not allowed to be modified by the
 DDI Editor as the information derives from the Journal DB.
 - Universes and releated sub-universes are listed in the Study Unit Editor.
 - The Study Unit Editor now always run in a view only mode.
 - Question import now support uppercase and lowercase v as variable name 
 reference. 
 - Question import now supports statement item and computation item
 - Sequence Flow now supports statement item and computation item
 - Now supports access to DDA database. Import of study level information from 
 SDSkriv.

Uninstalling
-------------------------------------------------------------------------------

 - Remove enviroment variable DDIEDITOR_HOME
 - Remove from enviroment variable: %DDIEDITOR_HOME%\resources\native-libs
