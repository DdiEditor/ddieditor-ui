What's New
-------------------------------------------------------------------------------
User interface
Item ID:    Description:
- Add pop-up menu point has been removed from Code table of the Code Scheme 
Editor. Use drag and drop of Cagetory instead to create new Code objects.  
- ControlConstructSchemes in Instrumentation view now supports labels.
- Import->Study Level Info does now import universes correctly.
- All views are refreshed after import. 
- Requestion for confirmation is given when File->Exit is chosen. 
- Multiple Question Item editor now supports Name attribute.
- Import questions as wiki text bug fix.
- Variable question view bug fix.
- SPSS import now has wizard.
